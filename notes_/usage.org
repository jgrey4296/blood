* CLI Usage
** sync
registers all profiles, finds modules, and installs module packages
** run
registers profiles, bootstraps,
sets up defaults,
activates profile,
inits packages
configs packages
** clean
registers profiles, deletes cache dirs
** report
registers profiles,
generates report on
- load order
- memory usage
- load time
- packages
- modules
- profiles
** stub
creates a new stub module/profile/package in a given location
